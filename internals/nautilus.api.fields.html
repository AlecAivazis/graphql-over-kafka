nautilus.api.fields.connection module¶

class nautilus.api.fields.connection.Connection(target, relationship='many', **kwds)[source]¶

Bases: graphene.core.types.field.Field

A field that encapsultes a connection with another GraphQL object.

Parameters:
  • target (string or ObjectType) – The target object type. If target is a SerivceObjectType then the remote data lookup is automated using the target object’s meta service attribute.
  • (string (relationship) – “one” or “many”): The kind of relationship that this connection encapsultes. If set to “many” the connection will result in a list whereas if it is set to “one” the connection will result in the object type itself.

Example

For an example, see the getting started guide.

resolve_service(instance, query_args, info)[source]¶

This function grab the remote data that acts as the source for this connection.
